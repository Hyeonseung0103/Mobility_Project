프로젝트 배경 및 목적 설명

오늘날 우리는 자동차를 굳이 타지 않더라도 가까운 거리라면 오토바이, 자전거, 킥보드 등과 같은 개인 이동장치를 어플 하나로 손쉽게 접근하여 이동할 수 있습니다.

하지만, 개인 이동장치의 사용이 증가함에따라 아래 그래프와 같이 개인 이동장치로 인한 교통사고 사례들이 해마다 꾸준히 증가하고 있습니다.







개인 이동장치는 자동차에 비해 안전 장치가 잘 구축되어 있지않기때문에 교통사고가 발생했을시 이용자가 입는 피해가 매우 큽니다.

또한, 개인 이동장치들의 교통법 위반으로 인해 차량 뿐만 아니라 보행자들마저 교통사고의 대상이 될 수 있습니다.

따라서, 본 프로젝트는 개인 이동장치의 교통법 위반 유형을 탐지하는 모델을 개발하여

동영상이나 이미지 내의 위반 유형을 탐지하고, 탐지 결과를 토대로 유형에 따른 처벌을 강화하여

사고의 발생건수와 피해정도, 위법 사례 등을 감소시키는 것을 목적으로 진행되었습니다.




사용 모델과 성능 설명

객체탐지에 사용된 모델은 Yolov8n 모델이고, 해당 모델은 0.72의 map50 수치를 기록했습니다.

객체탐지에서 주로 사용되는 map50이라는 지표는 모델이 예측한 객체 탐지 박스와 실제 객체가 존재하는 박스가 겹치는 비율이 0.5이상일 때(IoU 임계값 >= 0.5)

모델이 객체를 잘 검출해내면서도 불필요한 예측을 최소화하는 즉, 객체 탐지의 정확도를 나타내는 수치라고 표현할 수 있습니다.

map50은 0~1사이의 값을 가지고 수치가 1에 가까울수록 모델이 객체를 정확하게 탐지한다는 의미가 됩니다.

모델이나 성능에 대한 보다 구체적인 설명은 프로젝트 자료를 참고해주세요.

이제 아래의 가이드를 참고하여 객체 탐지를 진행해봅시다!




MP4 파일을 H264 코덱 형식으로 변환하기

크롬 브라우저에서는 MP4 파일이라고 할지라도 비디오 코덱이 H264 형식으로 이루어져 있어야 웹페이지에 출력할 수 있습니다.

따라서, 사용자의 파일이 H264 형식이 아닌 MP4 파일이라면 MP4 파일의 비디오 코덱을 "비디오 파일 to H264 변환" 이라는 버튼을 통해 변환시킬 수 있습니다.




1. "파일 선택" 버튼을 클릭하여 변환하고자 하는 MP4 파일을 선택합니다.




2. 파일 선택이 완료되었다면 "비디오 파일 to H264 변환" 버튼을 클릭하여 변환을 시작합니다.




3. 만약 MP4 파일이 아닌 다른 파일을 선택한 후 변환 버튼을 클릭하면 올바른 파일을 선택하라는 메시지가 출력됩니다.




4. 동영상은 길이에 따라 변환시간이 다르게 소요되며 길게는 몇분 후 변환된 이름의 파일이 자동으로 로컬 컴퓨터에 다운로드됩니다.




이미지 객체 탐지하기

이미지 객체 탐지도 동영상 파일 변환처럼 파일 선택 버튼을 통해 원하는 이미지를 선택합니다.




1. "이미지 객체탐지/초기화" 버튼을 클릭하면 모델이 이미지 내의 교통법 위반 유형을 탐지합니다.




2. 예측을 수행한 후, 예측 결과는 각 클래스를 어떻게 해석해야하는지의 정보와 함께 확인할 수 있습니다.

원본과 예측 이미지를 대조하며 모델이 예측한 클래스 번호, 번호에 매칭되는 위반 유형 이름, 총 위반 건수를 확인할 수 있습니다.




3. 이미지 객체탐지 버튼을 한번 더 클릭하면 페이지가 초기화됩니다.




영상 내의 객체 탐지하기

비디오(영상) 객체 탐지도 파일 선택 버튼을 통해 원하는 영상을 선택합니다.




1. "비디오 객체탐지/초기화" 버튼을 클릭하면 모델이 영상 내의 교통법 위반 유형을 탐지합니다.




2. 예측을 수행한 후, 예측 결과는 각 클래스를 어떻게 해석해야하는지의 정보와 함께 확인할 수 있습니다.

원본과 예측 영상을 대조하며 모델이 예측한 위반 유형 이름을 확인할 수 있습니다.

비디오는 여러개의 이미지가 합쳐진 형태로 이루어져있기때문에 같은 클래스가 중복되어 표현될 수 있습니다.

따라서, 이미지와는 달리 해당 영상 내 등장한 위반 유형만을 결과창에 출력합니다.




3. 비디오 객체탐지 버튼을 한번 더 클릭하면 페이지가 초기화됩니다.
